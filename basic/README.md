# Curso Angular (basic + medium)

Proyecto practico para aprender Angular 21 comparando cada tema con Vue.

## Scripts

```bash
npm start
npm test
npm run build
```

## Estructura del laboratorio

- `/basic` track basico
- `/medium` track medio
- `/:level/lessons/:slug` lecciones practicas con demo + comparacion Angular vs Vue
- `/:level/lessons/:slug/practice` playground con verificacion deterministica (100% tests)
- `docs/vue-to-angular-bridge.md` guia puente para expertos en Vue

## Modulos por nivel

### Basico

1. Signals y estado local
2. Templates y control flow
3. Input/Output entre componentes
4. Servicios y DI
5. Router (params/query/navigation)
6. Reactive Forms
7. HTTP + RxJS
8. Testing basico con Vitest

### Medio

1. Feature Architecture para SaaS
2. Signal Store Patterns
3. Router avanzado
4. HTTP Interceptors
5. Reactive Forms avanzados
6. Orquestacion RxJS
7. Performance y lazy rendering
8. Testing de integracion Angular
